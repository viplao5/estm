# 企业科技成果管理工作台 (ESTM)

## 🐳 项目简介

**企业科技成果管理工作台 (ESTM)** 是一个专门为企业设计的科技成果管理系统，旨在实现科技成果的全生命周期管理。本项目基于 [yudao-boot-mini](https://gitee.com/yudaocode/yudao-boot-mini) 框架进行开发，保留了其高效、灵活、可扩展的特性，同时针对科技成果管理业务进行了深度的模块化定制。

系统的核心以**研发项目**为纽带，关联科研平台、资质、核心技术人员及各类科技成果（如知识产权、技术秘密、论文著作等），构建了完整的数据链路。

---

## 🐨 技术栈

### 后端
- **框架**: Spring Boot
- **数据库**: MySQL
- **持久层**: MyBatis Plus
- **权限安全**: Spring Security + JWT
- **模块化**: 业务逻辑集成在 `yudao-module-bus` 模块中

### 前端
- **框架**: Vue 3 + TypeScript
- ** UI 组件**: Element Plus
- **构建工具**: Vite

---

## 🐼 核心业务模块

系统内置了以下 10 个核心业务管理模块：

| 模块名称 | 描述 |
| :--- | :--- |
| **研发项目管理** | 科技成果的源头，支持与及其关联的平台、资质、人员、成果进行全方位关联。 |
| **知识产权管理** | 统一管理专利、商标、软著等资产信息，支持官文扫描件上传与预览。 |
| **技术秘密管理** | 高级别的科研成果管理，包含严密的权限控制和保密级别设置。 |
| **论文与著作管理** | 记录企业的学术产出，支持多作者（核心技术人员）协作关系。 |
| **标准管理** | 追踪企业参与制订的行业、国家或国际标准。 |
| **科技奖励管理** | 记录企业及人员获得的各类科技荣誉和奖励。 |
| **产品与服务管理** | 将科技成果转化为实际的产品，关联所应用的技术秘密和知识产权。 |
| **科研平台管理** | 管理企业承载研发项目的物理或逻辑载体（如实验室、技术中心）。 |
| **核心技术人员管理** | 维护人才信息，多维度展示技术人员参与的项目和获取的成果。 |
| **资质管理** | 维护企业的各项认证、证书及其有效期。 |

## 🏛 系统架构

系统采用分层架构设计：
1. **用户交互层**: 基于 Vue3 的现代 Web 界面。
2. **应用服务层**: 集成了研发项目管理、成果管理等 10 大核心微服务逻辑。
3. **平台支撑层**: 提供认证授权、消息通知、文件存储、可视化报表等基础功能。
4. **基础设施层**: 支持私有云或公有云部署。

## 🚀 快速启动

### 1. 环境准备
- JDK 17+
- MySQL 5.7/8.0+
- Redis 5.0+
- Node.js 16+ & PNPM

### 2. 后端开发
1. 克隆代码后，在 `yudao-module-bus` 中查看业务逻辑。
2. 修改 `application-dev.yaml` 中的数据库和 Redis 连接。
3. 运行 `YudaoServerApplication` 启动程序。

### 3. 前端开发
1. 进入 `yudao-ui` 目录。
2. 执行 `pnpm install` 安装依赖。
3. 执行 `pnpm dev` 启动开发服务器。
